Ejecute el programa desde el directorio raíz(aquí se encuentran la carpeta src y target) utilizando el comando: 

java -cp target/PSP5Simpson-1.0-SNAPSHOT.jar com.ism.simpson.controller.AppController


La aplicación soporta que ingrese la ruta completa de un archivo llamado “numeros.txt”. Si no probee la ruta utilice el archivo provisto en la raíz del proyecto entregado. El programa leerá este archivo por defecto.
